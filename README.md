# ♻ Smart E-waste Inventory Management and Tracking System: Object-oriented Implementation of a CRUD Approach

## ℹ Description
This repository contains the final project files for CPE-009A Object-oriented Programming 1

## 🎯 Project Overview
The E-waste Inventory Management and Tracking System is a command-line based Python program developed to address the challenges of E-waste item classification, item registration, and data set consistency. It is designed to assist users in managing inventory, tracking inventory data, and statistical analysis of E-waste items.
